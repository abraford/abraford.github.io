Read with me

OK, so this isn't so good. This was my final project in Web Design 1. My daughter was only 9 months old when I started at MCAD, so when I wasn't working or doing home work, I was reading to her.

But I did learn a lot in the process of building this. The processbook format for designing anything is one that I still follow - this portfolio site took about 30 drawings to get to.

My background is as a print designer, where bigger is always better. This project was when I started to move away from that design ethic. It was also have been able to let go some print design cues that just don't work on the web, type wraps and the like.
